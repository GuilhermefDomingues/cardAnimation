# Flip Profile Card

Projeto desenvolvido com HTML e CSS que cria um card de perfil interativo com efeito flip. Ao passar o cursor sobre o card, a face frontal gira revelando informações do usuário e seus links de redes sociais.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Font Awesome

## ✨ Funcionalidades

- Card com efeito flip
- Foto de perfil
- Informações do usuário
- Redes sociais integradas
- Animações suaves
- Sem JavaScript
- Design moderno

## 📂 Estrutura do Projeto

```bash
Flip-Profile-Card/
│
├── index.html
├── css/
│   └── style.css
└── README.md
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.

Praticar efeitos de transformação 3D utilizando CSS para criar componentes modernos e interativos para portfólios, equipes e perfis de usuários.
